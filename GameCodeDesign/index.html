“优雅，永不过时”

游戏开发是使用最多设计模式的开发之一，其系统的复杂度高，且逻辑难以解耦。

# 包括下列内容：

  • 对设计模式的消化整理（主要基于Game Programming Patterns (opens new window)这本书）
  • 游戏泛用设计模式整理和实现
    • 如：游戏启动流程，上下文，UI等设计
  • 特定游戏模式的设计方式
    • 如：RTS相机，RPG中的换手等。
  • Unity里的异步、多线程和协同开发
