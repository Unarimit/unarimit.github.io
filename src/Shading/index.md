# 渲染技术

渲染决定了游戏的最终视觉呈现。本部分主要记录各种渲染技术和风格的实现原理及笔记。

## 包括下列内容：
- 渲染风格的研究，包括PBR（基于物理的渲染）和NPR（非真实感渲染）。
- 自然环境的模拟，如天空、云、雾效和地形的渲染。
- Unity渲染管线（URP）的学习与光照系统的应用。
- 屏幕后处理效果（Post Process）的实现。
